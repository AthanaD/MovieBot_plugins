## 识别名修正

> 基于MovieBot插件系统实现的识别名修正插件。

### 使用

#### 一、下载本插件

- 下载本插件，将autofix文件夹拷贝到插件目录。
- 重启MovieBot

#### 二、使用说明

- 插件无需任何配置。
- 自动识别下载的剧集片名是否正确，如有错误自动修正
- 插件依赖于MovieBot下载完成事件触发
- 不排除误删正常文件的可能，请谨慎使用
